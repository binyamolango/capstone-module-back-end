# 📖 Capstone Module Back End <a name="about-project"></a>

**Capstone Module Back End** is a rails API for reserving doctors.

## 🛠 Built With <a name="built-with"></a>
Ruby on Rails, Visual Studio Code

### Tech Stack <a name="tech-stack"></a>

- **Ruby on Rails**
- **PostgreSQL**

### Key Features <a name="key-features"></a>

- **Database implementation**
- **API endpoints**
- **User model**
- **Doctor model**

## 💻 Getting Started <a name="getting-started"></a>

### Setup <a name="setup"></a>

To use this project, Clone this repository to your desired folder by pasting this command in your command line interface:

  - `git clone https://github.com/binyamolango/capstone-module-back-end.git`

My front-end project link:

  - [git clone https://github.com/binyamolango/capstone-module-front-end](https://github.com/binyamolango/capstone-module-front-end)

### Prerequisites <a name="prerequisites"></a>

  Ruby on Rails
  Node
  Git

### Install <a name="install"></a>

To install the project's dependencies, run:

```
  bundle install
  yarn install
```

### Usage <a name="usage"></a>

To run the project, execute the following command:

```
  bin/dev
```

### Run tests <a name="run tests"></a>

To test, run:

```
  rspec
```
